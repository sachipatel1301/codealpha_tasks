# 🚀 Projects Included

## 1️⃣ Portfolio Website

A modern developer portfolio designed to showcase skills, projects, experience, and contact information.

### ✨ Highlights

* Animated hero section
* Typing text effect
* Scroll reveal animations
* Animated statistics counters
* Responsive design
* Mobile navigation menu
* Contact section
* Dark theme UI

### 🛠 Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript

## 2️⃣ Frame — Image Gallery

A responsive image gallery featuring category filtering, search functionality, favorites, bookmarks, and fullscreen lightbox viewing.

### ✨ Highlights

* Masonry gallery layout
* Lightbox viewer
* Category filters
* Live image search
* Favorites & bookmarks
* LocalStorage support
* Responsive design
* Authentication simulation

### 🛠 Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
* LocalStorage API

## 3️⃣ Lyrix — Premium Music Player

A feature-rich music player with playlist management, audio visualization, search, favorites, playback modes, and a premium UI experience.

### ✨ Highlights

* Audio playback controls
* Playlist management
* Music search
* Audio visualizer
* Favorites system
* Recently played history
* Theme switching
* Keyboard shortcuts
* Responsive design

### 🛠 Tech Stack

* HTML5
* CSS3

## ⚡ Quick Start

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/codealpha-frontend-projects.git
```

2. Navigate to any project folder

```bash
cd Portfolio-Website
```

or

```bash
cd Frame-Image-Gallery
```

or

```bash
cd Lyrix-Music-Player
```

3. Open `index.html` in your browser.

No installation, build tools, frameworks, or dependencies required.
